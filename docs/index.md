DaVinci Project Server Documentation
====================================

Welcome to the official documentation for the DaVinci Project Server.
This site provides installation instructions, architecture details,
troubleshooting guidance, and information for contributors.

Overview
--------

The DaVinci Project Server is a portable, self-contained deployment of
a DaVinci Resolve project library server. It includes:

- A PostgreSQL database containing Resolve project libraries
- A GUI and backend service for management
- Automated install, update, backup, and restore scripts
- Optional Docker Compose support
